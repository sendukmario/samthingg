### 2. Wallet PnL Chart

Get profit and loss chart data.

```
GET /api-v1/nova-wallet/wallet-pnl-chart (done)
```

**Query Parameters:**

- `wallet` (required): Wallet address
- `timeframe` (required): One of ["1d", "1w", "30d", "1y"]

**Response:**

```json
{

    "walletAddress": "string",
```
